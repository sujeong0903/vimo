# '비모' 비디오 인물 검색의 모든 것
> 얼굴 트래킹을 이용한 영상 속 등장인물 분석 및 검색 서비스



### 비모란?
비모는 얼굴 추적을 이용한 동영상 속 등장인물 분석 및 검색을 제공하여 사용자가 원하는 동영상을 빠르고 정확하게 찾아주고 등장인물의 등장시간, 등장 비율 등을 제공해 편리한 영상 시청을 돕는 APP입니다.
